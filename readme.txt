deepcore_v1.0

    DeepCore是一款超轻量级专为CNN批量训练量身打造的高度优化的核心计算库。

    支持的硬件：计算能力为5.0,5.2,6.0,6.1，7.0的NVIDIA GPU。


vdeepcore是deepcore的一个分支版本，专门针对volta优化的版本且仅支持volta GPU,由于专门针对tensor core进行了优化，因此数据结构差别很大，因此为简单以及避免代码过度膨胀，从volta开始会有一个新的分支版本且与之前的版本不兼容

2018/01/27:vdeepcore 1.1
    1 添加了池化操作
    2 内部添加了数十个针对小尺寸数据优化的卷积核矩阵乘法内核
    3 开放了数据几口，便于用户根据deepcore的数据结构添加自定义操作

现在仅支持64位win7系统，使用前请确保CUDA toolkit版本>=9.1,以及安装最新的驱动
